# Weekly NPS (2017)
def calcNPS(df):
    promoter  = df[df["Rating (Num)"] > 8]
    detractor = df[df["Rating (Num)"] < 7]
    nps = (promoter["Rating (Num)"].count() - detractor["Rating (Num)"].count()) / df["Rating (Num)"].count()
    return nps

df_weeks = df_2017.groupby("Week")
df_weeks.apply(lambda x: calcNPS(x)).plot(title="NPS (2017)")

Conclusions & Suggestions¶

  • The Summer Academy went well overall

  • Some locations need improvements, but we couldn't find a reason from pacing nor tracks.

  • We should collect other information, here're other elements we can consider:

  • Student demographics
  • Teachers

  • Locations
